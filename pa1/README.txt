Programming Assignment #1
-------------------------

These are the files for Programming Assignment #1.
This programming assignment was designed by the faculty of University of Chicago and does not belong to myself. 
The instructions can be found at https://classes.cs.uchicago.edu/archive/2021/fall/12100-1/pa/pa1/index.html


- sir.py: Skeleton code. You will write your code in this file.

- test_sir.py: The automated tests. See the programming assignment
  writeup for instructions on how to run them.

- pytest.ini and .pylintrc: Configuration files that you can safely ignore.

- tests/: A directory containing files specifying the tests.

- README.txt: This file.
